# AI Prompt Analysis

## Feladatértelmezés
A projektben használt AI-promptok célja az volt, hogy a meglévő kódbázist a beadási követelményekhez igazítsam, és megállapítsam, mely részek hiányoznak még.

## Rövid promptok és hatásuk
- „Szigorúan nézve, mi az amiket nem teljesít a project jelenlegi álapotában?”
  - Ezt követelmény-ellenőrző kérdésként értelmeztem.
  - Eredmény: a hiányzó repo-szintű dokumentációt és a demo adat hiányát azonosítottam.

- „készítsd el”
  - Ezt végrehajtási utasításként értelmeztem.
  - Eredmény: elkészült a docs mappa dokumentációval és a seed adatokkal.

## AI használati módszer
- A forráskód és a mappastruktúra alapján történt a követelményekhez illesztés.
- A dokumentációt a repo szerkezetéhez igazítva, külön fájlokba rendezve készítettem el.
- A demo adatokhoz olyan seedet hoztam létre, amely a meglévő auth/session logikával is használható.

## Megjegyzés
Ha a beadáshoz külön, formális promptnapló kell, ezt a fájlt ki lehet egészíteni konkrét promptokkal, válaszokkal és időbélyegekkel.
